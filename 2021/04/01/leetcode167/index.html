力扣167之双指针的妙处

题目描述

leetcode167–两数之和2:给定一个已按照 升序排列 的整数数组 numbers ，请你从数组中找出两个数满足相加之和等于目标数 target 。

函数应该以长度为 2 的整数数组的形式返回这两个数的下标值。numbers 的下标 从 1 开始计数 ，所以答案数组应当满足 1 <= answer[0] < answer[1] <= numbers.length 。

你可以假设每个输入只对应唯一的答案，而且你不可以重复使用相同的元素。
来源：力扣（LeetCode）
链接：https://leetcode-cn.com/problems/two-sum-ii-input-array-is-sorted

示例 1：    
输入：numbers = [2,7,11,15], target = 9
输出：[1,2]
解释：2 与 7 之和等于目标数 9 。因此 index1 = 1, index2 = 2 。

暴力法

方法1 暴力法 执行耗时:328 ms,击败了5.13% 的Java用户, 时间复杂度o(n^2)，重置导致多次遍历

//固定一个位置，从它后面找与它之和=target的位置
   public int[] twoSum(int[] numbers, int target) {
    int pre = 0;
    int later = 1;
    int length = numbers.length;
    while (pre < later){
        if (numbers[pre] + numbers[before] == target){
            return new int[]{pre + 1,later + 1};
        }else {
            later ++;
        }
        if (before >= length){
            //重置
            pre ++;
            later = pre + 1;
        }
    }
    return null;
}

二分法

方法2 二分查找 执行耗时:4 ms,击败了22.07% 的Java用户，时间复杂度
o(nlogn),和暴力法相比还是明显的，二分查找将n转为logn

      //二分查找
public int[] twoSum(int[] numbers, int target) {
    //前面和暴力法相同
    int length = numbers.length;
    int pre = 0;
    int later = length - 1;
    int middle;
    for (int i = 0; i < length; i++) {
        pre = i + 1;
        later = length - 1;
        while (pre <= later){
            middle = (pre + later) / 2;
            if (numbers[middle] == target - numbers[i]){
                return new int[]{i + 1,middle + 1};
            }else if(numbers[middle] > target - numbers[i]){
                later = middle - 1;
            }else {
                pre = middle + 1;
            }
        }
    }
    return null;
}

双指针法

方法2 双指针 执行耗时:1 ms,击败了93.03% 的Java用户，时间复杂度o(n),只遍历了一次数组

//双指针法,指针一前一后，如果指针两位置和小于target，左指针右移
public int[] twoSum(int[] numbers, int target) {
        //临界条件
        if (numbers.length <= 1) return null;
        int left = 0;
        int right = numbers.length - 1;
        while (left < right) {
            int sum = numbers[left] + numbers[right];
            if (sum == target) {
                return new int[]{left + 1,right + 1};
            }else if (sum > target){
                //如果大于taget,右指针左移
                right --;
            }else left ++;
        }
        return null;
    }
}

总结心得

不得不说双指针在这种处理有序数组的题里，真的太妙了,虽然这是一个easy题,但解法的不同带来的效率的不同真不禁让我感叹——算法的重要性！
