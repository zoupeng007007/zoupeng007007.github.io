力扣121 买卖股票的最佳时机

题目描述

给定一个数组 prices ，它的第 i 个元素 prices[i] 表示一支给定股票第 i 天的价格。你只能选择 某一天 买入这只股票，并选择在 未来的某一个不同的日子 卖出该股票。设计一个算法来计算你所能获取的最大利润。
返回你可以从这笔交易中获取的最大利润。如果你不能获取任何利润，返回 0 。

输入：[7,1,5,3,6,4]
输出：5
解释：在第 2 天（股票价格 = 1）的时候买入，在第 5 天（股票价格 = 6）的时候卖出，最大利润 = 6-1 = 5 。
     注意利润不能是 7-1 = 6, 因为卖出价格需要大于买入价格；同时，你不能在买入前卖出股票。

题目分析

其实就是求局部最小值和局部最大值差的问题，遍历时记录小值，并算出其后每个大值 - 小值，就是收益，记录更新最大收益即可

class Solution {
    public int maxProfit(int[] prices) {
        if(prices.length <= 1){
            return 0;
        }
        int maxEarn = 0;
        int minPrice = prices[0];
        for(int i = 1;i<prices.length;i++){
            maxEarn = Math.max(maxEarn,prices[i] - minPrice);
            minPrice = Math.min(minPrice,prices[i]);
        }
        return maxEarn;
    }
}

时间复杂度O(n)

122 买卖股票的最佳时机 II

题目描述

给定一个数组，它的第 i 个元素是一支给定股票第 i 天的价格。
设计一个算法来计算你所能获取的最大利润。你可以尽可能地完成更多的交易（多次买卖一支股票）。注意：你不能同时参与多笔交易（你必须在再次购买前出售掉之前的股票）。

输入: [7,1,5,3,6,4]
输出: 7
解释: 在第 2 天（股票价格 = 1）的时候买入，在第 3 天（股票价格 = 5）的时候卖出, 这笔交易所能获得利润 = 5-1 = 4 。
     随后，在第 4 天（股票价格 = 3）的时候买入，在第 5 天（股票价格 = 6）的时候卖出, 这笔交易所能获得利润 = 6-3 = 3 。

题目分析

保证每次都是卖出赚的，尽可能连续，一旦跌就不持有

class Solution {
    public int maxProfit(int[] prices) {
        //思路：保证每次都是赚的，尽可能连续
        //临界
        int length = prices.length;
        if(length <= 1){
            return 0;
        }
        int sum = 0;//记录最大利润
        int price = prices[0];//记录前一天价格
        for(int i = 1;i<length;i++){
            if(prices[i] > price){//持有
                sum += prices[i] - price;
            }
            price = prices[i];
        }
        return sum;
    }
}

贪心思想，求所有不相交区间长度和。

class Solution {
    public int maxProfit(int[] prices) {
        int ans = 0;
        int n = prices.length;
        for (int i = 1; i < n; i++) {
            ans += Math.max(0, prices[i] - prices[i - 1]);
        }
        return ans;
    }
}

时间复杂度O(n)
